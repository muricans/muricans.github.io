module options

Name Returns
message the message that module has
edit edit the message from that module
use whether you want to use that module or not <true|false>
channel the channel that module uses
everyone whether you want to allow everyone to use that module or not <true|false>
default default value for that module

custom command modules

When making a custom command, you can start the command message with '{module}' to turn it into a module command.

Name Returns
{mention} The user mentioned in the message. If no mention is provided it defaults to the user who sent the message.
{id} The id of the mentioned user. If there is no mention, it defaults to the user who sent the message.
{time} The current time that the bot is hosted in.
